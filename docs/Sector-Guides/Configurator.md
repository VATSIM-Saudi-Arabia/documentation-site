## What is the Configuartor?

The **Configurator** is a third-party, user-friendly application to enable controllers to semi-customize sector file packages with different themes, configure connection dialogs through an intuitive interface.

---

### Utilization

The application offers two main tools, **Theme Changer** and **EuroScope Profile**.

![Loading](imgs/conf.png#center)

Controllers are not required to install the application separately, as it is already included in the sector file package. Once the package has been extracted, the application can be found by navigating to the `OEJD\Plugins\Configurator` directory located within the `Plugins` folder. 

![Loading](imgs/exp.png#center)

From there, simply launch the `exe` file to access its configuration features.

---

#### Theme Changer

![Loading](imgs/thm.png#center)

The Theme Changer tool allows controllers to customize the visual appearance of the sector file according to their personal preferences.
There are two themes available to choose from:

- Realistic Indra ManagAir theme (Default).
- Dark Indra ManagAir theme.

To apply the theme, navigate to the sector file installation directory and choose the preferred theme then simply click `Apply`.

!!! note
    Make sure to restart EuroScope after applying the changes.

---

#### EuroScope Profile

![Loading](imgs/esc.png#center)

To improve the overall controller experience, this tool provides a simple and convenient connection dialog configurator. It allows controllers to configure their connection settings once and automatically apply them across all profiles within the package, eliminating the need to manually set up the same connection information for each individual profile.

To begin, navigate to the installation directory and complete the following fields with your VATSIM account information:

- Real Name (Refer to Code of Conduct A4(b))
- Certificate ID (CID)
- Password
- Rating

After entering the required information, click `Apply` to save the configuration. Finally, restart EuroScope for the changes to take effect.
