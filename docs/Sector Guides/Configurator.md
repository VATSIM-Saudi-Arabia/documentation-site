## What is the Configuartor?
Configurator is a third-party, user-friendly application to enable controllers to semi-customize sector file packages with different themes, configure connection dialogs through an intuitive interface in just a few clicks.

---

### Utilization

The application offers two main tools, **Theme Changer** and **EuroScope Profile**.

![Loading](imgs/conf.png#center)

Controllers are not required to install the application separately, as it is already included in the sector file package. Once the package has been extracted, the application can be found by navigating to the `OEJD\Plugins\Configurator` directory located within the `Plugins` folder. 

![Loading](imgs/exp.png#center)

From there, simply launch the `exe` file to access its configuration features.

---

#### Theme Changer

![Loading](imgs/thm.png#center)

Theme Changer tool allows controllers to switch the sector file visual appearance to their likings.
There are two themes available to choose from:

- Realistic Indra ManagAir theme (Default).
- Dark Indra ManagAir theme.
